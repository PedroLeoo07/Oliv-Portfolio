Habilidades

Desenvolvimento Web

O desenvolvimento web envolve a criação de sites e aplicações, divididos em frontend (interface do usuário com HTML, CSS, JavaScript) e backend (processamento de dados com Node.js). Os dados são armazenados em bancos de dados SQL como MySQL ou PostgreSQL, proporcionando experiências dinâmicas e funcionais na web.

Desenvolvimento Mobile

O desenvolvimento mobile com React Native e Expo permite criar aplicativos para Android e iOS com uma única base de código em JavaScript. O Expo oferece ferramentas e APIs prontas para uso, facilitando o processo sem configurações complexas. Com o Expo Go, é possível testar os aplicativos rapidamente em dispositivos reais ou emuladores.

UI/UX Design

UI/UX Design com Figma permite criar interfaces e melhorar a experiência do usuário de forma colaborativa e eficiente. O Figma facilita o design visual (UI) com ferramentas para layout e tipografia, e a criação de protótipos interativos para testar a navegação e usabilidade (UX). A colaboração em tempo real e a capacidade de criar designs responsivos tornam o processo mais ágil e dinâmico.

Adobe Photoshop

Tenho experiência básica no Adobe Photoshop, com habilidades em edição de imagens, como ajustes de cores e correção de imperfeições, além de design gráfico para criar logotipos e materiais visuais simples. Estou desenvolvendo minhas habilidades em arte digital, focando em ilustrações e manipulações criativas.
